# Overview

Marketplace Connect is a native Android app I built to strengthen my practical mobile engineering skills in building maintainable, user-focused products. My goal is to improve at designing real app flows, applying clean architecture ideas, and shipping reliable features end to end.

The app is a classified marketplace where users can create product listings, browse existing listings, search by keywords, filter by category, and sort results by newest or price. To use the app, launch to the home screen, tap the add button to create a listing, return to the list to browse, then open any item to view full details.

I created this app to practice turning core software engineering concepts into a complete Android experience: UI design, local data persistence, state handling, and separation of concerns with a ViewModel-based structure.

The YouTube demo will include a full walkthrough of the app running (create, search, filter, sort, and view details) plus a short explanation of the code structure and key classes.
